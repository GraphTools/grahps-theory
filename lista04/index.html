Exercise 4.1.2

Show that the following three statements are equivalent.

  1. $G$ is connected and has $n−1$ edges.
  2. $G$ is a forest and has $n−1$ edges.
  3. $G$ is a tree.

Prova

Caso $c \rightarrow a$. Se um grafo $G$ é uma árvore, então é conectado e $G$ possui $n−1$ arestas

Base

Dada uma árvore $G$ com $n=2$ e $m = n-1$ portanto $m=1$. Vamos supor que seja possível obter o mesmo resultado para uma árvore com $n-1$ vértices.

Hipótese de Indução

Para um grafo $G$ que é uma árvore com $n-1$ vértices, esta deverá ter então $n-2$ arestas.

Este resultado deve ser válido também para uma árvore de $n$ vértices.

Tese de Indução

Sendo $T$ uma árvore de $n$ vértices, esta possui então $n-1$ arestas.

Removendo um vértice $l$ de $T$ vamos obter $T'$, portanto $T' = T-l$.

Assim, $T'$ é um grafo conectado, pois, sendo $a$ e $z$ dois vértices de $T'$, existe um caminho de $a$ para $z$, e o mesmo ocorre em $T$. Este caminho em $T$ não contém $l$, pois em $T'$, $l$ não existe, e não seria possível haver um caminho entre $a$ e $z$ em $T'$ caso $l$ estivesse neste caminho e fosse removido, formando um grafo desconectado, o que é um absurdo.

Com isto, é evidente que $T'$ é uma árvore com $n-1$ vértices e $n-2$ arestas.

Caso $a \rightarrow b$. $G$ é uma floresta e contém $n-1$ arestas.

Vamos dizer que por contradição $G$ contém ciclos. Se removermos uma aresta de cada ciclo, a fim de formar um novo grafo $G'$, e de maneira que este permaneça conectado, e assim formando uma árvore, podemos dizer pelo item anterior que $G'$ contém $n-1$ arestas. Então $G$, deverá conter mais arestas que $G'$, porém é um absurdo que $G_n = n-1 \gt G'_n = n-1$. Com isto $G$ é acíclico, sendo ele uma floresta.

Caso $b \rightarrow c$. $G$ é uma floresta e tem $n-1$ arestas, portanto $G$ é uma árvore.

Numa primeira suposição, onde a floresta $G$ contivesse $n$ vértices e também $n$ arestas, já seria refutada por absurdo, pois um grafo com $m = n$ contém um ciclo, e uma floresta não contém ciclos.

Para isto vamos provar que se $G$ não contém ciclos, e que é uma árvore, ou seja um grafo conectado, como mostrado no primeiro caso.

Se $G$, por contradição, fosse um grafo desconectado, onde $k$ a quantidade de suas componentes, conectadas e acíclicas (assim como $G$), então este contém $G_1, G_2,..., G_k$, onde $k \geq 2$.

Para tornar este grafo conectado devemos adicionar 1 aresta conetando uma componente a outra. Porém estas arestas adicionadas devem ser no total de $k-1$ para não fechar um ciclo.

Com a adição destas arestas, para tornar $G$ conectado, o resultado da soma das arestas de $G$ é de $n-1+k-1 = m$, o que é um absurdo, desde que este deve conter $n-1$ arestas.

Exercise 4.2.11 - Prüfer code

Let $K_n$ be the labelled complete graph with vertex set $ \{1, 2, . . . , n\} $, where $n \geq 3$. With each spanning tree $T$ of $K_n$ one can associate a unique sequence $(t1,t2,...,tn−2)$, known as the Prüfer code of $T$, as follows. Let $s_1$ denote the first vertex (in the the ordered set $(1,2,...,n)$ which is a leaf of $T$, and let $t_1$ be the neighbour of $s_1$ in $T$. Now let $s_2$ denote the first vertex which is a leaf of $T-s_1$, and $t_2$ the neighbour of $s_2$ in $T−s_1$. Repeat this operation until $t_n−2$ is defined and a tree with just two vertices remains. (If $n \leq 2$, the Prüfer code of $T$ is taken to be the empty sequence.)

  1. List all the spanning trees of $K_4$ and their Prüfer codes.
  2. Show that every sequence $(t_1,t_2,...,t_{n−2})$ of integers from the set $ \{1,2,...,n\} $ is the Prüfer code of a unique spanning tree of $K_n$.
  3. Deduce Cayleys Formula (see Theorem 4.8$^2$ )
