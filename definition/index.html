Graph
A graph $G$ consists of $V(G)$, a set of vertices (often also called nodes) and $E(G)$, a set of edges
Simple graph
Has no loops
The degree of a vertex
Is the number of edges that are incident of $v$ vertex
Neighborhood
Set of all vertices sharing an edge with $v$
The handshaking lemma
In any graph $\sum d(v) = 2|E|$
$K_n$
Is a complete graph on $n$ vertices
Bipartite graph
Has two sets of vertices, where in a set there are no edges between it members. Is allowed to has vertices without edges. Notation: $G[X,Y]$
Isolated graph
No edges
Complete bipartite graph
All vertices of a X partition shares edge with all vertices of the partition Y and vice-versa. On $n$ vertices $G[X,Y]$, with $|X| = r$ and $|Y| = s$ number of elements, the number of edges are: $r.s \leq \frac{n^2}{4}$
The star graph
It is a bipartite graph where one of the sets has just one member.
Grid graph
Isomorphic Graphs
Two graphs $G_1$ and $G_2$ that there is a 1-1 onto a bijection function from $V_{(G_1)}$ to $V_{(G_2)}$ preserving adjacency.
Hypercube
$Q_n := \{$ $0-1$ vector (nodes), such that any two nodes have an edge between them exactly when they differ in $1$ coordinate$\}$
Planar Graph
Is a graph that can be drawn in the plane such that edges can meet only their corresponding vertices.
Subgraph
A subgraph $F$ of a graph $G$ is a graph that uses a subset of the vertices of $G$, such that: $V_{(F)} \leq V_{(G)}$ and $E_{(F)} \leq E_{(G)}$.
Incidence Matrix
Is the $n \times m$. $M_G := (m_{ve})$. $m_ve$ is the number of times $(0,1,2,...)$ that a vertex $v$ and edge $e$ are incident.
Adjacency Matrix
Is the $n \times n$ . $A_G := (a_{uv})$. a_{uv} is the number of edges joining the vertices $u$ and $v$. In case of loops, it counts two edges.
Isolated vertex
A vertex with degree equals to $0$;
Eigenvalue
An Eigenvalue of a matrix $M$ is a complex number given by $M_x = \lambda_x$, where $x$ is a vector, called an eigenvector of $M$ and $\lambda \in \mathbb{C}$ (is a complex number)
Walk
A walk in a graph $G$ is a sequence of nodes (vertices) $w = v_0e_1,v_1e_2,v_2e_3,\cdots,v_{l-1}e_l,v_le_{l+1}$ where the terms are alternating vertices and edges of $G$ (not necessarily distinct) such that $v_{i-1}$, and $v_i$ are the end points of $e_i, $e_i, 1 \leq i \leq l$
Path
A path is a walk which does not repeat any vertices.
Cycle
Diâmetro maior distância entre quaisquer dois vértices Distância é o menor caminho entre dois vertices Diametro infinito quando o grafo é desconectado
